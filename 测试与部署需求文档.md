# 长沙大学生兼职平台 — 测试与部署需求文档

> **角色**: Agent-3 测试+运维工程师
> **工作目录**: `deploy/`, `docker/`, 项目根目录测试配置

---

## 一、环境搭建（第1-3周）

### 1.1 云服务器配置
- 领取免费云服务器（ECS 1核2G），配置安全组
- 安装 Docker、Nginx、MySQL 8.0、Redis 7.0

### 1.2 网络与安全配置
- 配置 Nginx 反向代理 + HTTPS（Let's Encrypt）

### 1.3 容器化部署
- 编写 `docker-compose.yml`（一键启动所有服务）
- 编写后端 `Dockerfile`

### 1.4 CI/CD 配置
- 配置 Jenkins CI/CD 流水线
- 部署 Spring Boot 后端（单实例）

### 1.5 数据备份
- 配置 MySQL 每日全量备份脚本（OSS）+ 每小时增量备份
- OSS 版本控制开启

---

## 二、测试框架搭建（第2-4周）

### 2.1 API自动化测试
- 编写 API 自动化测试脚本（Postman Collection Runner / pytest）

### 2.2 性能测试
- 编写 JMeter 性能测试脚本（模拟1万用户并发）

### 2.3 代码质量扫描
- 配置 SonarQube 代码质量扫描

---

## 三、功能测试（第4-12周，随开发进行）

### 3.1 全流程闭环测试
- 学生注册 → 认证 → 投递 → 面试 → 签约 → 打卡 → 结算 → 评价

### 3.2 薪资异常场景测试
- 企业余额不足时发起发放
- 支付回调丢失（幂等性）
- 重复发放
- 个税计算正确性

### 3.3 电子签约测试
- 协议生成 + 哈希存证验证

### 3.4 投诉处理测试
- 资金冻结 + 补偿流程

---

## 四、合规测试（第10-12周）

### 4.1 数据安全验证
- 验证敏感数据 AES-256 加密存储（查数据库确认密文）
- 验证脱敏展示（查 API 返回确认手机号/身份证脱敏）

### 4.2 审计日志验证
- 验证操作审计日志完整性（查 t_audit_log）

### 4.3 五流合一验证
- 验证数据关联：agreement_id / salary_flow_id / invoice_id / job_id / trace_id

### 4.4 等保基础项验证
- HTTPS + TLS1.2 + 数据备份 + 权限隔离

---

## 五、安全测试（第10-12周）

### 5.1 注入攻击测试
- SQL 注入测试（所有查询接口）
- XSS 攻击测试（岗位描述/简历输入）

### 5.2 权限测试
- 未授权访问测试（学生调企业接口，企业调管理员接口）

### 5.3 令牌安全测试
- JWT 令牌伪造测试

### 5.4 限流测试
- 接口限流测试（快速连续请求验证429）

### 5.5 支付安全测试
- 支付接口重放攻击测试

---

## 六、兼容性测试（第11-12周）

### 6.1 小程序端
- 微信/支付宝最新版

### 6.2 PC端
- Chrome/Edge/Safari 近2年版本

### 6.3 H5端
- iOS Safari / Android Chrome

---

## 七、监控与运维（第8-12周）

### 7.1 云监控告警
- CPU ≥80%, 内存 ≥90%, 磁盘 ≥85%

### 7.2 服务监控
- 部署 Uptime Kuma 监控后端服务

### 7.3 故障演练
- 编写故障演练脚本（模拟 DB 崩溃，RTO ≤1小时验证）

### 7.4 应急响应
- 编写《安全事件应急响应手册》

## 十二、关键约束

1. **成本约束**: MVP 阶段使用免费云服务器
2. **时间约束**: 故障恢复时间（RTO）≤1小时
3. **合规约束**: 审计日志保留3年以上
4. **安全约束**: 所有管理后台接口 IP 白名单校验
